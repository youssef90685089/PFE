## 1. PROBLEM STATEMENT

### 1.1 Project Context

As part of the digital transformation of academic and professional processes, managing internships and academic projects remains a major challenge for higher education institutions and companies. Traditional management methods, often based on Excel spreadsheets and email exchanges, present several significant limitations:

- **Administrative Inefficiency**: Manual tracking of applications, evaluations, and assignments generates considerable workload for competent services.
- **Lack of Traceability**: The absence of a centralized system prevents data consolidation and makes performance analysis difficult.
- **Insufficient Matching**: Manual matching between candidates, projects, and supervisors does not allow optimal resource optimization.
- **Fragmented Communication**: Candidates are rarely informed about the status of their applications, creating constant dissatisfaction and inquiries.
- **Absence of Intelligence**: Assignment decisions are made based on subjective criteria rather than objective analysis of skills and requirements.

### 1.2 Identified Problem

**How to modernize and automate the complete internship and project management cycle, from application submission to final evaluation, while integrating artificial intelligence to optimize candidate-project-supervisor matching?**

This problem breaks down into several operational questions:

1. How to centralize all data related to candidates, projects, and supervisors?
2. How to automate the application and selection process?
3. How to integrate an AI system for intelligent matching?
4. How to ensure real-time communication between stakeholders?
5. How to generate analytical reports for decision-making?

### 1.3 General Objective

Design and develop a complete web application called **SIPMS** (Smart Internship & Project Management System) to digitize the entire academic internship and project management process, with an artificial intelligence component for automatic candidate-project matching.

### 1.4 Specific Objectives

| Objective | Description |
|-----------|-------------|
| OG1 | Centralize user management (candidates, supervisors, administrators) |
| OG2 | Automate application submission and evaluation process |
| OG3 | Develop an online quiz system with automatic grading |
| OG4 | Implement an AI-based recommendation engine for matching |
| OG5 | Create an analytical dashboard for statistics |
| OG6 | Implement a real-time notification system |
